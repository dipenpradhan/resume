APPS

Expense Tracker

Click here to see on play store

An android app for keeping a track of daily expenses. It features a calendar view for ease of use, and provides a simplistic approach to entering, viewing and photo-tagging expenses with support for multiple currencies with live conversion rates.

The app includes a custom calendar grid implementation with swiping gestures and animations, photo capture and storage for tagging expenses with receipts or bills, and a list based view for viewing year-wise, month-wise, and day-wise expenses.

This app was built by me as a freelance programmer for Snoops Co., Australia, based on the requirements and designs provided.

Restaurant Mobile Ordering app

Developed as part of a 3 member team (as a full time Software Engineer at Movivation Labs Pvt. Ltd.) this application is used by customers to order food from the client's restaurants.

It has a clean, easy to use interface, allowing for quickly picking food items from a menu card and placing orders (triggering SMS and email notifications).

My role on the team was in developing the android application and part of the back end - RESTful web services built using the Java Spring framework and MySQL database.

Drunk Man

Click here to download

A game for android devices where the player is challenged with to make a drunk man walk in a straight line. Built entirely from scratch using OpenGL ES, the game involves balancing a drunk man walking on a lonely road, whose drunkenness increases with time.

This game was built by me as a freelance programmer for Neurolinx Software Technologies Pvt. Ltd., Mumbai-India based on the requirements, graphic assets and design mockups provided.

The entire project, inclusive of developing the game engine, movement and rendering logic was handled by me.

Bot Wars - Academic Project - B.E.(Computers)

Click here to download More info and source code at:https://code.google.com/p/bot-wars/

A side-scrolling 2-D game for Android devices, built with a team of 3 students as an academic project for the final year of our Bachelor's Degree in Computer Engineering. It features single and multiplayer gameplay.

The core gameplay was built using AndEngine and Box2D libraries, along with a custom client-server style multiplayer implementation based on UDP (for WiFi and 3G) and Bluetooth RFCOMM sockets.

Bot wars has received over 600 downloads and over 25 code forks from google code and github open source repositories.
